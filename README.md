# MasterQ 游戏

## 游戏介绍
MasterQ 是一款支持围棋和五子棋的桌面游戏应用，具有现代化的界面和丰富的功能。

## 功能特点

### 游戏模式
- **围棋**：19x19 标准棋盘，支持完整的围棋规则
- **五子棋**：15x15 标准棋盘，支持完整的五子棋规则

### 核心功能
- **游戏控制**：开始游戏、暂停/继续、悔棋、重做
- **棋谱管理**：记录棋谱、回放棋谱
- **界面控制**：窗口拖拽、全屏显示、主题切换（亮色/深色）
- **音频系统**：8-bit 风格音效，包括落子、按钮点击等音效

### 视觉效果
- **现代化界面**：简洁美观的用户界面
- **主题切换**：支持亮色和深色两种主题
- **自定义图标**：使用独特的游戏图标

## 如何运行

### 系统要求
- Windows 10 或更高版本
- .NET 8.0 或更高版本

### 运行方式
1. 下载并解压游戏文件
2. 运行 `MasterQ.exe` 文件启动游戏

## 游戏操作

### 基本操作
- **开始游戏**：选择游戏模式后点击"开始游戏"按钮
- **落子**：在棋盘上点击鼠标左键
- **悔棋**：点击"悔棋"按钮
- **暂停/继续**：点击"暂停/继续"按钮
- **全屏**：点击窗口右上角的全屏按钮
- **主题切换**：点击窗口右上角的主题切换按钮

### 游戏模式选择
在游戏控制区域的下拉菜单中选择"围棋"或"五子棋"模式。
